datetime::operator==

Tests for equality.

Synopsis 
constexpr bool
operator==(
    const datetime& rhs) const;
Description

Two datetimes are considered equal if all of its individual components are equal. This function works for invalid datetimes, too.

Exception safety

No-throw guarantee.
