metadata::table

Returns the name of the virtual table the column belongs to.

Synopsis 
string_view
table() const;
Description

If the table was aliased, this will be the name of the alias (e.g. in "SELECT * FROM employees emp", table() will be "emp").

This is optional information - it won't be populated unless the connection executing the query has meta_mode() == metadata_mode::full.

Exception safety

No-throw guarantee.

Object lifetimes

The returned reference is valid as long as *this is alive and hasn't been assigned to or moved from.
