statement::valid

Returns true if the object represents an actual server statement.

Synopsis 
bool
valid() const;
Description

Calling any function other than assignment on a statement for which this function returns false results in undefined behavior.

Returns false for default-constructed statements.

Exception safety

No-throw guarantee.
