row::operator== (1 of 3 overloads)

Equality operator.

Synopsis 
bool
operator==(
    const row& lhs,
    const row& rhs);
Description

The containers are considered equal if they have the same number of elements and they all compare equal, as defined by field_view::operator==.

Exception safety

No-throw guarantee.

Complexity

Linear in lhs.size() and rhs.size().
