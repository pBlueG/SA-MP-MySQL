field::as_date (1 of 2 overloads)

Retrieves a reference to the underlying date value or throws an exception.

Synopsis

Exception safety

Strong guarantee. Throws on type mismatch.

Exceptions

Type

Thrown On

Object lifetimes

The returned reference is valid as long as *this is alive and no function that invalidates references is called on *this.
