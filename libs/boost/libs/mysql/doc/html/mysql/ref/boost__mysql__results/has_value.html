results::has_value

Returns whether the object holds a valid result.

Synopsis 
bool
has_value() const;
Description

Having this->has_value() is a precondition to call all data accessors. Objects populated by connection::query, connection::execute_statement or their async counterparts are guaranteed to have this->has_value() == true.

Exception safety

No-throw guarantee.
