segments_encoded_ref::url

Return the referenced url.

Synopsis 
url_base&
url() const;
Description

This function returns the url referenced by the view.

Example 
url u( "/path/to/file.txt" );

assert( &u.encoded_segments().url() == &u );
Exception Safety

Throws nothing.
