make_pct_string_view

Return a valid percent-encoded string.

Objects of this type behave like a string_view and have the same interface, but offer an additional invariant: they can only be constructed from strings containing valid percent-escapes. Attempting construction from a string containing invalid or malformed percent escapes results in an exception.
