grammar::recycled_ptr::operator= (1 of 2 overloads)

Assignment.

Synopsis 
recycled_ptr&
operator=(
    recycled_ptr&& other);
Description

If other references an object, ownership is transferred including a reference to the recycle bin. After the move, the moved-from object is empty.

Effects 
this->release()
Postconditions 
&this->bin() == &other->bin()
Exception Safety

Throws nothing.

Parameters

Name

Description

other

The pointer to move from
