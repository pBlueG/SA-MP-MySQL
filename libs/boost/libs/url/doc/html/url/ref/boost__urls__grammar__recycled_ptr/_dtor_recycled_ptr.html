grammar::recycled_ptr::~recycled_ptr

Destructor.

Synopsis 
~recycled_ptr();
Description

If this is not empty, shared ownership of the pointee is released. If this was the last reference, the object is returned to the original recycle bin.

Effects 
this->release();
