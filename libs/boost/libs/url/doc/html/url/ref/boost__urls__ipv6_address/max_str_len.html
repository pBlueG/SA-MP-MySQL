ipv6_address::max_str_len

The number of characters in the longest possible IPv6 string.

Synopsis 
static
constexpr std::size_t max_str_len = 49;
Description

The longest IPv6 address is: 

ffff:ffff:ffff:ffff:ffff:ffff:ffff:ffff
