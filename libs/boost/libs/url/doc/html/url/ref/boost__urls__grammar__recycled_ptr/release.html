grammar::recycled_ptr::release

Release the referenced object.

Synopsis 
void
release();
Description

If this references an object, it is released to the referenced recycle bin. The pointer continues to reference the same recycle bin.

Postconditions 
this->empty()
Exception Safety

Throws nothing.
