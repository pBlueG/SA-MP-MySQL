param::param (1 of 4 overloads)

Constructor.

Synopsis 
param();
Description

Default constructed query parameters have an empty key and no value.

Example 
param qp;
Postconditions 
this->key == "" && this->value == "" && this->has_value == false
Complexity

Constant.

Exception Safety

Throws nothing.
