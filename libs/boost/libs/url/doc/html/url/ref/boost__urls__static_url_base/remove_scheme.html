static_url_base::remove_scheme

(Inherited from url_base)

Remove the scheme.

Synopsis 
url_base&
remove_scheme();
Description

This function removes the scheme if it is present.

Example 
assert( url("http://www.example.com/index.htm" ).remove_scheme().buffer() == "//www.example.com/index.htm" );
Postconditions 
this->has_scheme() == false && this->scheme_id() == scheme::none
Complexity

Linear in this->size().

Exception Safety

Throws nothing.

BNF 
URI           = scheme ":" hier-part [ "?" query ] [ "#" fragment ]
Specification
  • 3.1. Scheme (rfc3986)
